PostgreSQL + Prometheus + Grafana
=========

Набор ролей позволяет установить PostgreSQL, Prometheus, Grafana и пару экспортеров node_exporter c prometheus-postgres-exporter, при этом первый и последний устанавливаются из внешних репозиториев, а для установки остального требуется предварительно скачать пакеты или архивы и поместить их в директории `Files` соответствующих ролей. Имена файлов внести в `start.yml`.  
Хранение больших файлов внутри роли скорее является антипаттерном, но таковы были граничные условия выполнения задачи.  
По возможности лучше выложить файлы на http-«шару», например, развернув nginx и внеся примерно такие изменения:
* добавить переменную
```
  vars:
    grf_paht: "http://192.168.1.3/files/grafana_12.3.0_19497075765_linux_amd64.deb" # http путь до пакета
```
* изменить playbook
```
- name: install grafana on local
  ansible.builtin.apt:
    deb: "{{ grf_paht }}"
  register: status_packed
```
Возможная конфигурация для web-сервера есть в директории nginx.  
Данная установка не предполагает настройку сбора метрик, только дефолтные установки.
